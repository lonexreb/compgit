# Decisions

Dated decision log. One file per decision. File name: `YYYY-MM-DD-short-slug.md`.

Every decision page:

1. Names the alternative(s) considered.
2. Says what we picked.
3. Explains why in a sentence or two.
4. Notes any condition that would make us revisit it.
