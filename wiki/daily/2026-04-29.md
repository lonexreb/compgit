## Pre-Phase-2b prep landed

Four of the five prep tasks from [`../../NEXT-TO-DO.md`](../../NEXT-TO-DO.md) shipped today, each as its own conventional commit:

1. **`feat(shared-ts): accept optional baseURL`** — `fetchContributionsCollection` (via the args object) and `fetchViewerLogin` (third optional positional). `GITHUB_GRAPHQL_URL` is now exported. 43 vitest tests pass (was 41); two new cases assert the URL is honoured.
2. **`feat(shared-swift): accept optional baseURL`** — `GitHubClient` actor init takes `baseURL: URL = GitHubClient.defaultEndpoint`. `swift build` clean.
3. **`feat(shared-swift): add Tokens.swift`** — `CompgitTokens` namespace with OKLCH triples for dark + light, type sizes in pt (CSS rem × 16), 4-pt spacing, motion durations in seconds. Pure data — no SwiftUI / AppKit / UIKit imports per the rule.
4. **`feat(ios|macos): XcodeGen project.yml + README`** — both surfaces have a YAML source-of-truth for their Xcode project. Targets, bundle IDs, App Group, Keychain Sharing, BackgroundModes (iOS), App Sandbox + hardened runtime (macOS), and SPM dep on `packages/shared-swift` are all declared. `*.xcodeproj/` is gitignored. A future agent runs `brew install xcodegen && cd apps/ios && xcodegen generate` once Xcode lands.
